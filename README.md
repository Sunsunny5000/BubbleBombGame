# BubbleBomb

基于 Java Swing 的本地双人泡泡堂游戏初版，使用 JDK 11 编译，不依赖第三方库。

## 启动

在项目根目录执行：

在 VS Code 中直接运行 `com.bubblebomb.game.GameStart`。

## 操作

- 玩家一：`W A S D` 移动，`B` 放置泡泡，`V` 遥控引爆。
- 玩家二：方向键移动，`Enter` 放置泡泡，`/` 遥控引爆。
- 标题页：方向键左右选择地图、上下设置箱子密度，`Enter` 开始。
- 标题页：`H` 游戏说明，`I` 道具说明。
- 游戏中：`Esc` 返回标题。
- 游戏中：`P` 暂停；暂停后可继续、重开当前回合或返回标题。
- 结束页：`R` 或 `Enter` 重新开始。

箱子全部清除后进入两分钟决胜；最后一分钟地图从边缘对称缩小，最后三十秒显示红色警告。

比赛采用先胜 3 回合制。每回合胜者获得 1 分，平局不加分；回合结算页按 `Enter` 或 `N` 进入下一回合。

开放花园的加速区和泥地迷宫的泥地区均采用成片布局。加速效果离开地形后保留 2 秒，泥地减速离开后保留 1 秒；停留在地形内会不断刷新持续时间。
